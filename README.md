# P2-AI-Travel-Planner-Agent

An AI-powered travel planning assistant designed to help users create personalized travel itineraries and recommendations.

## Overview
This project demonstrates an intelligent agent-based approach to travel planning. It focuses on understanding user preferences and generating useful travel suggestions in a simple and structured way.

## Key Features
- Personalized travel planning
- AI-assisted itinerary generation
- Travel recommendations based on user preferences
- Clean and modular project structure

## Tech Stack
- Python
- AI/agent-based workflow
- LLM-powered conversational logic

## Project Highlights
- Practical example of an AI planning agent
- Good for portfolio and learning purposes
- Easy to extend with additional travel features

## Getting Started
1. Clone the repository
2. Install the required dependencies
3. Run the project locally
